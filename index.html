Welcome to memex-explorer’s documentation.¶

The goal of the memex-explorer project is to provide users with an easy-to-use, fully-featured web interface to use for gathering, analyzing, and graphing web crawl data. With memex-explorer you can register data URIs that contain your data to be monitored. You can also add plots using the data from your data URIs. In future versions, all of this graphing and streaming data capabilities will be available to you in an easy-to-use, customizable dashboard interface.

To get started, read our user’s guide. For more in-depth information about the project, refer to our developer’s guide and our API guide.

memex-explorer is built by Continuum Analytics, with grants and support from the NASA Jet Propulsion Laboratory, Kitware, and the NYU Polytechnic School of Engineering
